<h1 align="center"> Readme Design Kit </h1>

This README design kit consists of various elements to enhance your README profile, including:  

Contribution Graph, Stats Cards, Language Cards, Repository Details Cards, Coding Platform Stats Cards, Profile View Counters, Quotes and Jokes Cards, Lines and Waves, Decorations and Animations.

Simply click on a title to expand the dropdown menu.  

To use these elements, copy the provided code below each section and replace **"Mayur-Pagote"** with your GitHub username.

## Glowing Neon Border

Add a vibrant glowing neon border around your GitHub README profile sections to create a modern, eye-catching effect.

### CSS Code

```css
.glowing-neon-border {
  position: relative;
  padding: 20px;
  border-radius: 10px;
  background-color: #111;
  color: #fff;
  font-family: 'Arial', sans-serif;
  font-weight: bold;
  overflow: hidden;
  z-index: 0;
}

.glowing-neon-border::before {
  content: "";
  position: absolute;
  top: -2px;
  left: -2px;
  right: -2px;
  bottom: -2px;
  border-radius: 12px;
  background: linear-gradient(45deg, #00f, #0ff, #0f0, #ff0, #f00, #f0f, #00f);
  background-size: 400% 400%;
  animation: neonGlow 20s linear infinite;
  filter: blur(4px);
  opacity: 0.7;
  z-index: -1;
}

@keyframes neonGlow {
  0% {
    background-position: 0% 50%;
  }
  50% {
    background-position: 100% 50%;
  }
  100% {
    background-position: 0% 50%;
  }
}
